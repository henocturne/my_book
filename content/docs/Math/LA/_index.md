# 高等代数
## 王萼芳
- author：王萼芳
- title：高等代数 第五版
- publisher: 高等教育出版社

根据《高等代数（第五版）》的目录及内容简介，本书的内容可以分为以下几个核心部分：

1.  **多项式理论**：主要探讨数域、一元多项式的整除、最大公因式、因式分解以及多元多项式等概念。
2.  **线性代数基础**：涵盖了行列式的计算与性质、线性方程组的解法（消元法）、矩阵的运算与秩，以及二次型的标准形与正定性。
3.  **抽象空间与线性变换**：深入讨论线性空间的结构（维数、基、坐标）、线性变换的矩阵表示、特征值与特征向量，以及欧几里得空间、双线性函数与辛空间等进阶课题。
4.  **矩阵的深入理论**：包含$\lambda$-矩阵、若尔当（Jordan）标准形以及相关的几何理论。
