在第七章“线性变换”中，为了严谨地描述线性变换及其代数性质，使用了涉及变换、矩阵、特征值及子空间等多个方面的数学符号。以下是根据来源内容总结的符号列表：

### 1. 线性变换与空间基本符号
*   **\\( \mathscr{A}, \mathscr{B}, \mathscr{C}, \dots \\)**：通常使用花体拉丁字母表示线性空间 \\( V \\) 上的**线性变换**。
*   **\\( V \\)**：表示**线性空间**（通常为 \\( n \\) 维）。
*   **\\( P \\)**：表示线性空间所属的**数域**。
*   **\\( \alpha, \beta, \gamma, \xi, \eta \\)**：通常使用希腊字母表示空间中的**向量**。
*   **\\( k, a \\)**：通常使用拉丁字母表示数域 \\( P \\) 中的**标量**。
*   **\\( \mathscr{A}(\alpha) \\) 或 \\( \mathscr{A}\alpha \\)**：表示向量 \\( \alpha \\) 在线性变换 \\( \mathscr{A} \\) 下的**像**。

### 2. 特定的线性变换符号
*   **\\( \mathscr{E} \\)**：**单位变换**（恒等变换），满足 \\( \mathscr{E}(\alpha) = \alpha \\)。
*   **\\( 0 \\)**：**零变换**，满足 \\( 0(\alpha) = \mathbf{0} \\)。
*   **\\( \mathscr{K} \\)**：**数乘变换**（由数 \\( k \\) 决定），满足 \\( \mathscr{K}(\alpha) = k\alpha \\)。
*   **\\( \mathscr{D} \\)**：**微分变换**（求微商），定义为 \\( \mathscr{D}(f(x)) = f'(x) \\)。
*   **\\( \mathscr{I} \\)**：**积分变换**，定义为 \\( \mathscr{I}(f(x)) = \int_0^x f(t) dt \\)。
*   **\\( \Pi_\alpha \\)**：**射影变换**（如沿向量 \\( \alpha \\) 的内射影或对平面的射影）。
*   **\\( \mathscr{R}_x \\)**：关于平面的**反射变换**。
*   **\\( \mathscr{S}_a \\)**：**平移变换**，例如在多项式空间中定义为 \\( f(\lambda) \to f(\lambda + a) \\)。

### 3. 线性变换的运算符号
*   **\\( \mathscr{A} + \mathscr{B} \\)**：线性变换的**和**，定义为 \\( (\mathscr{A} + \mathscr{B})(\alpha) = \mathscr{A}(\alpha) + \mathscr{B}(\alpha) \\)。
*   **\\( \mathscr{A}\mathscr{B} \\)**：线性变换的**乘积**（复合映射），定义为 \\( (\mathscr{A}\mathscr{B})(\alpha) = \mathscr{A}(\mathscr{B}(\alpha)) \\)。
*   **\\( k\mathscr{A} \\)**：线性变换的**数量乘法**。
*   **\\( -\mathscr{A} \\)**：线性变换 \\( \mathscr{A} \\) 的**负变换**。
*   **\\( \mathscr{A}^{-1} \\)**：线性变换的**逆变换**（若存在）。
*   **\\( \mathscr{A}^n \\)**：线性变换的 **\\( n \\) 次幂**。
*   **\\( f(\mathscr{A}) \\)**：线性变换的**多项式**。

### 4. 矩阵、基与相似符号
*   **\\( \varepsilon_1, \varepsilon_2, \dots, \varepsilon_n \\)**：线性空间 \\( V \\) 的一组**基**。
*   **\\( A, B \\)**：线性变换在特定基下对应的**矩阵**。
*   **\\( X \\)**：两组基之间的**过渡矩阵**。
*   **\\( A \sim B \\)**：表示矩阵 \\( A \\) **相似**于矩阵 \\( B \\)。
*   **\\( E \\)**：**单位矩阵**。
*   **\\( (x_1, \dots, x_n)^T \\)**：向量在某组基下的**坐标列向量**。

### 5. 特征值、子空间与标准形符号
*   **\\( \lambda, \lambda_0 \\)**：表示线性变换或矩阵的**特征值**。
*   **\\( |\lambda E - A| \\)**：矩阵 \\( A \\) 的**特征多项式**。
*   **\\( \text{tr } A \\)**：矩阵 \\( A \\) 的**迹**（主对角线元素之和）。
*   **\\( V_{\lambda_0} \\)**：属于特征值 \\( \lambda_0 \\) 的**特征子空间**。
*   **\\( V^{\lambda_i} \\)**：属于特征值 \\( \lambda_i \\) 的**根子空间**。
*   **\\( \mathscr{A}V \\)**：线性变换 \\( \mathscr{A} \\) 的**值域**（像空间）。
*   **\\( \mathscr{A}^{-1}(\mathbf{0}) \\)**：线性变换 \\( \mathscr{A} \\) 的**核**（零空间）。
*   **\\( W \\)**：通常表示**不变子空间**（\\( \mathscr{A} \\)-子空间）。
*   **\\( \mathscr{A} |_W \\)**：线性变换 \\( \mathscr{A} \\) 在其不变子空间 \\( W \\) 上的**诱导变换**（限制）。
*   **\\( V_1 \oplus V_2 \oplus \dots \oplus V_s \\)**：子空间的**直和**。
*   **\\( J(\lambda_0, k) \\)**：\\( k \\) 阶**若尔当块**。
*   **\\( \text{diag}(A_1, \dots, A_s) \\)**：以矩阵块为对角元素的**准对角矩阵**。
