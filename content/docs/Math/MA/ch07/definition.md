根据源文件《第七章 数项级数》，本章定义并使用了关于级数收敛性、无穷乘积以及各种特殊求和法的数学符号。以下是这些符号的分类总结：

### 1. 级数与部分和基础符号
*   **\\( \sum_{n=1}^\infty a_n \\)**：表示**无穷级数**，其中 \\( a_n \\) 称为**通项**或一般项。
*   **\\( S_n = \sum_{k=1}^n a_k \\)**：级数的**第 \\( n \\) 个部分和**。
*   **\\( \sum_{n=1}^\infty a_n = S \\)**：当部分和序列 \\( \{S_n\} \\) 收敛于 \\( S \\) 时，称 \\( S \\) 为**级数的和**。
*   **\\( \{a_n\} \\)**：表示实数序列。
*   **\\( |S_m - S_n| \\)**：用于 **Cauchy 准则**中表示部分和之间的差值。

### 2. 级数运算与变换符号
*   **\\( \sum_{n=1}^\infty (\lambda a_n + \mu b_n) \\)**：级数的**线性组合**。
*   **(7.2) \\( Abel \\) 变换**：\\( \sum_{i=1}^n a_i b_i = \sum_{i=1}^{n-1} (a_i - a_{i+1}) B_i + a_n B_n - a_1 B_0 \\)，其中 \\( B_i \\) 是 \\( b_i \\) 的部分和。
*   **\\( \sum_{n=0}^\infty c_n \\)**：两个级数 \\( \sum a_n \\) 与 \\( \sum b_n \\) 的 **Cauchy 乘积**，其中 \\( c_n = \sum_{i+j=n} a_i b_j \\)。
*   **\\( a_n^+ \\) 与 \\( a_n^- \\)**：级数通项的**正部与负部**，定义为 \\( a_n^+ = \frac{a_n + |a_n|}{2}, a_n^- = \frac{|a_n| - a_n}{2} \\)，用于讨论级数的重排。

### 3. 特殊函数与无穷乘积符号
*   **\\( \zeta(s) = \sum_{n=1}^\infty \frac{1}{n^s} \\)**：**Riemann-Zeta 函数**。
*   **\\( \prod_{n=1}^\infty p_n \\)**：**无穷乘积**。
*   **\\( P_n = \prod_{k=1}^n p_k \\)**：无穷乘积的**部分乘积**。
*   **\\( \sin x = x \prod_{l=1}^\infty [1 - (\frac{x}{l\pi})^2] \\)**：正弦函数的无穷乘积表示。
*   **\\( \sinh x \\) 与 \\( \cosh x \\)**：双曲正弦与双曲余弦函数的无穷乘积表示。

### 4. 广义求和与极限符号
*   **\\( \sum_{n=0}^\infty a_n = S (A) \\)**：在 **Abel 意义下可求和**，表示 \\( \lim_{x \to 1^-} \sum a_n x^n = S \\)。
*   **\\( \sum_{n=1}^\infty a_n = S (C) \\)**：在 **Cesàro 意义下可求和**，表示部分和的算术平均值 \\( \sigma_n = \frac{1}{n} \sum S_k \\) 的极限为 \\( S \\)。
*   **\\( \sum_{j=1}^\infty a_{ij} = \alpha_i \\) 关于 \\( i \\) 一致收敛**：用于讨论级数求和与取极限顺序的可交换性。

### 5. 其他辅助符号
*   **\\( \bigvee_\alpha^\beta(f) \\)**：虽然在第七章主要讨论数项级数，但在引理 7.1.1 的证明中提到了函数在区间上的性质，不过该符号更常见于第十三章的有界变差讨论。
*   **\\( S(\alpha) = \int_1^\alpha a(x) dx \\)**：在积分判别法的证明中，用于表示连续化后的部分和函数。
