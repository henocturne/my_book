根据提供的源文件《第八章 函数项级数》，该章节定义并使用了多类关于函数列、函数项级数以及幂级数的数学符号。以下是这些符号的分类总结：

### 1. 收敛与一致收敛符号
*   **\\( \{g_n(x)\} \\) 或 \\( g_n \\)**：定义在区间 \\( I \\) 上的**函数列**。
*   **\\( g(x) \\) 或 \\( g \\)**：函数列的**极限函数**。
*   **\\( g_n \to g \\)**：表示函数列在区间上**逐点收敛**。
*   **\\( g_n \rightrightarrows g \\)**：表示函数列在区间 \\( I \\) 上**一致收敛**。
*   **\\( \sup_{x \in I} |g_n(x) - g(x)| \\)**：用于刻画一致收敛的**上确界范数**（或最大偏差）。
*   **内闭一致收敛**：若函数列在区间 \\( I \\) 的任何闭子区间 \\( J \subset I \\) 上都一致收敛，则称其在 \\( I \\) 中内闭一致收敛。

### 2. 函数项级数符号
*   **\\( \sum_{n=1}^\infty f_n(x) \\)**：**函数项级数**。
*   **\\( S_n(x) = \sum_{k=1}^n f_k(x) \\)**：级数的**第 \\( n \\) 个部分和**。
*   **\\( S(x) \\)**：级数的**和函数**。
*   **\\( M_n \\)**：**Weierstrass 判别法**（M-判别法）中的常数项，满足 \\( |f_n(x)| \le M_n \\)。
*   **\\( B_n(x) = \sum_{k=1}^n b_k(x) \\)**：在讨论 Dirichlet 判别法或 Abel 判别法时使用的函数项级数部分和。

### 3. 幂级数与泰勒展开符号
*   **\\( \sum_{n=0}^\infty a_n x^n \\)**：以 0 为中心的**幂级数**。
*   **\\( R \\)**：幂级数的**收敛半径**。
*   **\\( \rho = \varlimsup_{n \to \infty} |a_n|^{1/n} \\)**：用于 **Cauchy-Hadamard 公式**计算收敛半径的项。
*   **\\( R_n(x) \\)**：Taylor 展开式的**余项**。
*   **\\( C_\alpha^n \\) 或 \\( \binom{\alpha}{n} \\)**：**广义组合数**，定义为 \\( \frac{\alpha(\alpha-1)\dots(\alpha-n+1)}{n!} \\)。
*   **\\( (n)!! \\)**：**双阶乘**。

### 4. 特殊函数与多项式符号
*   **\\( \zeta(s) = \sum_{n=1}^\infty \frac{1}{n^s} \\)**：**Riemann-Zeta 函数**。
*   **\\( B_n \\)**：第 \\( n \\) 个 **Bernoulli 数**（伯努利数），通过 \\( x/(e^x-1) \\) 的展开式定义。
*   **\\( B_n(t) \\)**：**Bernoulli 多项式**。
*   **\\( P_n(t) \\)**：**Legendre 多项式**。

### 5. 逼近与特殊构造符号
*   **\\( \varphi(x) \\) 与 \\( \psi(t) \\)**：常用于构造处处连续但处处不可导函数的**锯齿函数**。
*   **\\( \Phi(x) \\)**：用于 Taylor 展开系数任意性证明中的**特定平滑函数**。
*   **鼓包函数**：一种支集受限的光滑函数 \\( \phi(x) \\)，满足在特定范围内取值为 1，其余部分平滑过渡到 0。
