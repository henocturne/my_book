根据源文件《第十四章 微分形式的积分》，该章节各小节下的定理、命题、引理及推论整理如下：

### **§14.1 欧氏空间中的微分形式**
*   **命题 14.1.1**：设 \\( D \subset \mathbb{R}^n \\) 为区域，\\( X \\) 为 \\( D \\) 中的连续向量场，则 **\\( X \\) 为保守场当且仅当沿路径积分与路径选取无关**，而只依赖于路径的端点。
*   **推论 14.1.2**：设 \\( \omega \\) 为区域 \\( D \\) 中的连续 1-形式，则 **\\( \omega \\) 为某个函数的全微分当且仅当它沿 \\( D \\) 中任何闭曲线积分为零**。

### **§14.2 微分形式之间的运算**
*   **命题 14.2.1 (Poincaré 引理)**：设 \\( D \\) 为 \\( \mathbb{R}^n \\) 中的**凸区域**，则 \\( D \\) 中的**闭形式必为恰当形式**。

### **§14.3 Gauss-Green 公式**
*   **引理 14.3.1 (单位分解)**：设 \\( K \\) 为 \\( \mathbb{R}^n \\) 中的有界闭集，\\( \{V_\alpha\} \\) 为 \\( K \\) 的有限开覆盖，则存在 \\( \mathbb{R}^n \\) 中的**光滑函数族 \\( \{\phi_\alpha\} \\)（称为单位分解）**，满足在 \\( K \\) 上之和恒为 1 且其支集包含在对应的开集中。
*   **定理 14.3.2 (Gauss-Green 公式)**：设 \\( \Omega \\) 为 \\( \mathbb{R}^n \\) 中的正则区域，\\( \eta \\) 为 \\( \bar{\Omega} \\) 中 \\( C^1 \\) 的 \\( (n-1) \\)-形式，则有：**\\( \int_{\Omega} d\eta = (-1)^n \int_{\partial \Omega} \eta \\)**。
*   **引理 14.3.3**：给出了 \\( C^1 \\) 函数 \\( f \\) 和 \\( C^2 \\) 映射 \\( \varphi \\) 下，关于 **Jacobi 行列式的一个微分恒等式**，用于证明 Gauss-Green 公式。
*   **定理 14.3.4 (散度定理)**：设 \\( \Omega \\) 为 \\( \mathbb{R}^n \\) 中的正则区域，\\( X \\) 为 \\( \bar{\Omega} \\) 中的 \\( C^1 \\) 向量场，则其**散度在区域上的积分等于其沿边界单位外法向量的通量**：\\( \int_{\Omega} \text{div } X dx = \int_{\partial \Omega} X \cdot \vec{n} d\sigma \\)。
*   **定理 14.3.5 (平均值公式)**：设 \\( f \\) 为 \\( D \\) 中的**调和函数**（即 \\( \Delta f = 0 \\)），则 \\( f \\) 在某点的值等于其在以该点为球心的**球体上的平均值**。

### **§14.4 不动点定理和毛球定理**
*   **引理 14.4.1**：设 \\( \psi: D \to \mathbb{R}^n \\) 为连续向量值函数，则存在**光滑向量值函数 \\( \rho \\) 可以在 \\( D \\) 上任意逼近 \\( \psi \\)**。
*   **引理 14.4.2**：设 \\( \rho: D \to \mathbb{R}^n \\) 为 \\( C^2 \\) 向量值函数，如果当 \\( x \\) 在单位球面上时 \\( \rho(x) = x \\)，则 **\\( \rho \\) 在区域 \\( D \\) 内必有零点**。
*   **定理 14.4.3 (Brouwer 不动点定理)**：设 \\( D \\) 为 \\( \mathbb{R}^n \\) 中的闭球，\\( \varphi: D \to D \\) 为连续映射，则 **\\( \varphi \\) 必有不动点**。
*   **定理 14.4.4 (毛球定理)**：**偶数维球面上的连续切向量场必有零点**（形象地说，无法平滑地梳理偶数维毛球上的毛）。
