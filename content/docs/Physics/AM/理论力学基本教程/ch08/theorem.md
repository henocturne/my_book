在《理论力学基本教程》第八章“拉格朗日动力学”中，核心理论通常以“原理”、“方程”或“守恒律”的形式呈现。以下是按照小节整理的具有定理或引理性质的核心规律：

### §8-1 有势系的拉格朗日方程
*   **拉格朗日方程定理**：受完整约束的有势系，其真实运动满足 \\( s \\) 个二阶微分方程（即拉格朗日方程）：\\( \frac{d}{dt} \frac{\partial L}{\partial \dot{q}_a} - \frac{\partial L}{\partial q_a} = 0 \\)。该方程是哈密顿原理中作用量 \\( S \\) 取极值的必要条件。

### §8-2 广义动量积分和广义能量积分
*   **广义动量守恒定理（可遗坐标定理）**：若系统的拉格朗日函数 \\( L \\) 不显含某个广义坐标 \\( q_a \\)（即 \\( q_a \\) 为可遗坐标），则与该坐标对应的广义动量 \\( p_a \\) 守恒，即 \\( p_a = \frac{\partial L}{\partial \dot{q}_a} = \text{常量} \\)。
*   **广义能量守恒定理**：若系统的拉格朗日函数 \\( L \\) 不显含时间 \\( t \\)，则系统的广义能量 \\( H \\) 保持不变，即 \\( H = \sum_{\alpha=1}^s p_a \dot{q}_a - L = \text{常量} \\)。
*   **齐次函数欧拉定理（引理性质）**：在分析动能和广义能量结构时，利用欧拉定理证明 \\( \sum \frac{\partial T_2}{\partial \dot{q}_a} \dot{q}_a = 2T_2 \\) 和 \\( \sum \frac{\partial T_1}{\partial \dot{q}_a} \dot{q}_a = T_1 \\)，从而推导出 \\( H = T_2 - T_0 + V \\)。

### §8-3 一般形式的拉格朗日方程
*   **一般完整系动力学定理**：对于包含非有势力的完整系统，其运动方程由 \\( T \\) 和广义力 \\( Q_a \\) 表达为：\\( \frac{d}{dt} \frac{\partial T}{\partial \dot{q}_a} - \frac{\partial T}{\partial q_a} = Q_a \\)。

### §8-4 多自由度系统在稳定平衡位置附近的小振动
*   **简正模式分解定理**：多自由度线性系统的微小振动通解可以表示为若干个具有确定频率（简正频率）和特定振幅比例（简正模式）的谐振动的叠加。
*   **特征方程定理**：简正频率由特征方程（频率方程） \\( | -m\omega^2 + k_{ij} | = 0 \\) 确定。
*   **简正坐标解耦定理**：通过线性变换引入简正坐标 \\( \xi_i \\)，可以将原本耦合的运动方程组改写为一组相互独立的谐振子方程。

### §8-5 对称性与守恒律（诺特定理的体现）
*   **空间平移对称性定理**：若系统对空间平移具有不变性，则系统在该方向上的总动量守恒。
*   **空间转动对称性定理**：若系统对绕某轴的转动具有不变性，则系统对该轴的角动量守恒。
*   **时间平移对称性定理**：若系统具有时间对称性（\\( L \\) 不显含 \\( t \\)），则系统的广义能量守恒。

### §8-6 广义势 带电粒子的拉格朗日方程 耗散函数
*   **耗散函数能损定理**：在受线性性质摩擦阻力作用的系统中，系统的机械能耗散速率等于耗散函数 \\( F \\) 的两倍，即 \\( \frac{d}{dt}(T+V) = -2F \\)。

### §8-7 相对论力学中质点的拉格朗日函数
*   **相对论能量守恒律**：在有势场中运动的相对论质点，若 \\( L \\) 不显含时间，则相对论哈密顿函数（总能量） \\( H = \frac{m_0 c^2}{\sqrt{1 - v^2/c^2}} + V \\) 守恒。
