### 一、 主要内容概述
本章主要在**相空间**中研究力学系统的运动，引入了**哈密顿方法**（哈密顿表述）。内容从**勒让德变换**出发，将拉格朗日方程转化为**正则方程**（一阶微分方程组），并引入了**正则变量**、**相空间**和**哈密顿函数**等核心概念。随后详细讨论了**泊松括号**及其性质、**正则变换**理论，以及求解动力学问题的**哈密顿-雅可比方程**。

### 二、 定义整理
（注：原文中定义通常随正文叙述给出，此处按 9.1.1 形式统一整理其核心物理概念）

*   **定义 9.1.1 广义动量**：与广义坐标 \\( q_a \\) 对应的物理量 \\( p_a = \partial L / \partial \dot{q}_a \\)。
*   **定义 9.1.2 勒让德变换**：通过构造新函数 \\( g = ux - f \\)，将独立变量从 \\( (x, y) \\) 变换为 \\( (u, y) \\) 的数学方法。
*   **定义 9.1.3 哈密顿函数**：定义为 \\( H = \sum_{\alpha=1}^s p_a \dot{q}_a - L \\)，是表征系统动力学性质的特征函数。
*   **定义 9.1.4 正则变量**：\\( s \\) 个广义坐标 \\( q_a \\) 和 \\( s \\) 个广义动量 \\( p_a \\) 的统称。
*   **定义 9.1.5 相空间**：由 \\( 2s \\) 个正则变量 \\( (q_a, p_a) \\) 组成的 \\( 2s \\) 维空间。
*   **定义 9.1.6 相轨迹**：相空间中代表系统运动状态的点随时间变化的路径。
*   **定义 9.1.7 广义能量积分**：在 \\( H \\) 不显含时间 \\( t \\) 时，\\( H = \text{常量} \\) 的积分。
*   **定义 9.1.8 广义动量积分**：若 \\( q_a \\) 为可遗坐标（\\( H \\) 不显含 \\( q_a \\)），则 \\( p_a = \text{常量} \\) 的积分。
*   **定义 9.2.1 泊松括号**：两个关于正则变量的函数 \\( \varphi \\) 和 \\( \psi \\) 的特定运算形式，见公式 (9.2.1)。
*   **定义 9.2.2 基本泊松括号**：正则变量彼此之间的泊松括号，如 \\( [q_a, p_\beta] = \delta_{\alpha\beta} \\)。
*   **定义 9.2.3 泊松定理**：若两个力学量是守恒量，则它们的泊松括号也是守恒量。
*   **定义 9.3.1 正则变换**：保持正则方程形式不变的正则变量之间的坐标变换 \\( (q, p) \to (Q, P) \\)。
*   **定义 9.3.2 生成函数（母函数）**：确定正则变换关系的新旧坐标混合函数，如 \\( F_1, F_2 \\) 等。
*   **定义 9.3.3 哈密顿主函数**：哈密顿沿着真实运动路线计算的作用量 \\( S \\)，常作为正则变换的母函数。
*   **定义 9.3.4 哈密顿-雅可比方程**：母函数 \\( S \\) 必须满足的一阶偏微分方程，见公式 (9.3.26)。

### 三、 公式整理

#### §9-1 正则方程
*   **(9.1.1)** 拉格朗日方程：\\( \frac{d}{dt} \frac{\partial L}{\partial \dot{q}_a} - \frac{\partial L}{\partial q_a} = 0 \\)
*   **(9.1.11)** 勒让德变换新函数：\\( g = ux - f \\)
*   **(9.1.14)** 变换关系式：\\( x = \frac{\partial g}{\partial u}, \frac{\partial f}{\partial y} = -\frac{\partial g}{\partial y} \\)
*   **(9.1.19)** 哈密顿函数定义：\\( H = \sum_{\alpha=1}^s p_a \dot{q}_a - L \\)
*   **(9.1.23)** **正则方程（哈密顿方程）**：\\( \dot{q}_a = \frac{\partial H}{\partial p_a}, \dot{p}_a = -\frac{\partial H}{\partial q_a} \\)
*   **(9.1.24)** 时间偏导关系：\\( \frac{\partial H}{\partial t} = -\frac{\partial L}{\partial t} \\)
*   **(9.1.25)** 哈密顿函数的一般形式：\\( H = T_2 - T_0 + V \\)
*   **(9.1.30)** 广义能量积分：\\( H(q_a, p_a, t) = \text{常量} \\)
*   **(9.1.34)** 广义动量积分：\\( p_a = \alpha_a = \text{常量} \\)

#### §9-2 泊松括号
*   **(9.2.1)** **泊松括号定义**：\\( [\varphi, \psi] = \sum_{\alpha=1}^s \left( \frac{\partial \varphi}{\partial q_a} \frac{\partial \psi}{\partial p_a} - \frac{\partial \varphi}{\partial p_a} \frac{\partial \psi}{\partial q_a} \right) \\)
*   **(9.2.3)** 基本泊松括号：\\( [q_a, p_\beta] = \delta_{\alpha\beta} \\)
*   **(9.2.14)** 力学量随时间变化率：\\( \frac{dW}{dt} = \frac{\partial W}{\partial t} + [W, H] \\)
*   **(9.2.16)** 泊松括号形式的正则方程：\\( \dot{q}_a = [q_a, H], \dot{p}_a = [p_a, H] \\)
*   **(9.2.18)** 力学量守恒必要条件：\\( \frac{\partial W}{\partial t} + [W, H] = 0 \\)
*   **(9.2.32)** 泊松定理证明恒等式：\\( \frac{d[f, g]}{dt} = 0 \\)

#### §9-3 正则变换 哈密顿-雅可比方程
*   **(9.3.8)** 正则变换条件：\\( \sum p_a dq_a - \sum P_a dQ_a + (K - H) dt = dF_1 \\)
*   **(9.3.9)** 第一类母函数变换关系：\\( p_a = \frac{\partial F_1}{\partial q_a}, P_a = -\frac{\partial F_1}{\partial Q_a} \\)
*   **(9.3.10)** 新旧哈密顿函数关系：\\( K = H + \frac{\partial F_1}{\partial t} \\)
*   **(9.3.15)** 第二类母函数变换关系：\\( p_a = \frac{\partial F_2}{\partial q_a}, Q_a = \frac{\partial F_2}{\partial P_a} \\)
*   **(9.3.18)** 作用量变分关系：\\( \delta S = \sum p_a \delta q_a - \sum p_{\alpha0} \delta q_{\alpha 0} \\)
*   **(9.3.26)** **哈密顿-雅可比方程**：\\( \frac{\partial S}{\partial t} + H(q_a, \frac{\partial S}{\partial q_a}, t) = 0 \\)
*   **(9.3.31)** 哈密顿主函数全微分：\\( dS = \sum p_a dq_a - H dt \\)
*   **(9.3.32)** 动量与能量的导数表示：\\( \frac{\partial S}{\partial q_a} = p_a, \frac{\partial S}{\partial t} = -H \\)
