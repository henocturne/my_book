根据提供的源代码，第一章《复数和复变函数》中定义和使用的主要符号及其含义总结如下：

### **1. 复数基础定义与集合**
*   **\\( z \\)**：表示一个复数，可定义为有序实数对 \\( (x, y) \\)。
*   **\\( x \\) 或 \\( \text{Re } z \\)**：复数 \\( z \\) 的**实部** (Real part)。
*   **\\( y \\) 或 \\( \text{Im } z \\)**：复数 \\( z \\) 的**虚部** (Imaginary part)。
*   **\\( i \\)**：**虚单位** (Imaginary unit)，定义为 \\( (0, 1) \\)，满足 \\( i^2 = -1 \\)。
*   **\\( \mathbb{C} \\)**：**复数域**，由所有复数组成的集合。
*   **\\( \mathbb{R} \\)**：**实数域**，由所有实数组成的集合。
*   **\\( 0 \\)**：复数零，即 \\( (0, 0) \\)。
*   **\\( -z \\)**：复数 \\( z \\) 的**相反数**，即 \\( (-x, -y) \\)。

### **2. 共轭、模与代数运算**
*   **\\( z^\* \\) 或 \\( \bar{z} \\)**：\\( z \\) 的**共轭复数** (Complex conjugate)，定义为 \\( x - iy \\)。
*   **\\( |z| \\)**：复数 \\( z \\) 的**模** (Modulus)，即自原点到点 \\( (x, y) \\) 的距离 \\( \sqrt{x^2 + y^2} \\)。

### **3. 几何表示与极坐标**
*   **\\( (r, \theta) \\)**：复数的**极坐标**表示，其中 \\( x = r \cos \theta, y = r \sin \theta \\)。
*   **\\( r \\)**：同复数的模 \\( |z| \\)。
*   **\\( \theta \\) 或 \\( \arg z \\)**：复数 \\( z \\) 的**辐角** (Argument)。
*   **\\( e^{i\theta} \\)**：依据 **Euler 公式**定义的纯虚数指数函数：\\( e^{i\theta} = \cos \theta + i \sin \theta \\)。
*   **\\( \mathbb{E} \\)**：**扩充的复平面**，包含有限复平面 \\( \mathbb{C} \\) 和无穷远点 \\( \infty \\)。
*   **\\( \infty \\)**：**无穷远点**，其模大于任何正数，辐角不定。

### **4. 复数序列与极限**
*   **\\( \{z_n\} \\)**：**复数序列**，\\( z_n = x_n + iy_n \\)。
*   **\\( \lim_{n \to \infty} z_n = z \\)**：表示序列 \\( \{z_n\} \\) **收敛**于极限 \\( z \\)。
*   **\\( \varlimsup_{n \to \infty} x_n \\) (或 \\( \text{lim sup} \\))**：实数序列的**上极限**。
*   **\\( \varliminf_{n \to \infty} x_n \\) (或 \\( \text{lim inf} \\))**：实数序列的**下极限**。
*   **\\( \epsilon \\)**：在极限定义中表示任意小的正数。
*   **\\( N(\epsilon) \\)**：与 \\( \epsilon \\) 相关的正整数，用于定义序列的收敛性。

### **5. 区域与复变函数**
*   **\\( G \\)**：复平面内的**区域** (Region)。
*   **\\( C \\) 或 \\( \partial G \\)**：区域 \\( G \\) 的**边界** (Boundary)。
*   **\\( \bar{G} \\)**：**闭区域**，定义为 \\( G + C \\)。
*   **\\( w = f(z) \\)**：定义在区域 \\( G \\) 上的**复变函数**。
*   **\\( u(x, y) \\)**：复变函数 \\( f(z) \\) 的**实部**（二元实函数）。
*   **\\( v(x, y) \\)**：复变函数 \\( f(z) \\) 的**虚部**（二元实函数）。
