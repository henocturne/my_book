根据源文件《第三章 复变积分》的内容，现按小节整理其中的所有定理、引理及具有定理性质的核心公式：

### **§ 3.1 复变积分**
本节主要定义复变积分及其性质（如线性性质、路径加性、逆向性、估值性质等），**未出现**明确标注为“定理”或“引理”的内容。

### **§ 3.2 Cauchy 定理**
本节是全章的核心，包含多个基础引理和重要定理：
*   **定理 3.1 (Cauchy 定理)**：如果函数 \\( f(z) \\) 在有界闭区域 \\( \bar{G} \\) 中解析，则沿其边界 \\( C \\) 的积分 \\( \oint_C f(z)dz = 0 \\)。
*   **引理 3.1**：设 \\( K_1, K_2, \dots \\) 是复平面内一系列嵌套的有界闭区域（\\( K_1 \supset K_2 \supset \dots \\)），则至少存在一个点 \\( x_0 \in \bigcap K_n \\)。
*   **引理 3.2**：设 \\( \alpha \\) 和 \\( \beta \\) 为复常数，\\( \gamma \\) 为任意闭合三角形围道，则 \\( \oint_\gamma (\alpha + \beta z)dz = 0 \\)。
*   **引理 3.3**：对于有界单连通区域内的可求长曲线 \\( \gamma \\)，总存在内接于 \\( \gamma \\) 的折线 \\( \Gamma \\) 使其积分逼近 \\( \gamma \\) 上的积分。
*   **复变积分变形定理**（推论）：解析函数在区域内的积分值与路径无关。
*   **定理 3.2**：在有界单连通区域内，解析函数 \\( f(z) \\) 的不定积分 \\( F(z) = \int_{z_0}^z f(\zeta)d\zeta \\) 也是解析函数，且满足 \\( F'(z) = f(z) \\)。

### **§ 3.3 两个有用的引理**
本节专门介绍了在计算积分极限时常用的两个引理：
*   **引理 3.4（小圆弧引理）**：给出函数在奇点附近的小圆弧积分极限值，结果与圆弧张角及函数在该点的局域性质有关。
*   **引理 3.5（大圆弧引理）**：给出函数在无穷远点邻域内的大圆弧积分极限值。

### **§ 3.4 Cauchy 积分公式**
本节阐述了解析函数值与边界积分的关系：
*   **有界区域的 Cauchy 积分公式**：解析函数区域内任意一点的值可由边界积分完全决定。
*   **均值定理**（解析函数的特殊性质）：解析函数在某点的函数值等于以该点为圆心的任一圆周上函数值的平均值。
*   **无界区域的 Cauchy 积分公式**：将公式推广至包含无穷远点的无界区域情形。

### **§ 3.5 解析函数的高阶导数**
*   **解析函数的高阶导数公式**：证明解析函数在其解析区域内具有任何阶导数，并给出各阶导数的积分表示法。
*   **结论**：一个复变函数在其解析区域内任何阶导数都存在，且仍为该区域内的解析函数。

### **§ 3.6 Cauchy 型积分和含参量积分的解析性**
*   **Cauchy 型积分的解析性**：证明在分段光滑曲线上连续的函数构成的 Cauchy 型积分是曲线外点的解析函数。
*   **定理 3.3（含参量积分的解析性）**：若 \\( f(t, z) \\) 是连续且对 \\( z \\) 解析的函数，则其变限积分 \\( F(z) = \int_a^b f(t, z)dt \\) 也是解析函数，且求导与积分可交换次序。

### **§ 3.7 Poisson 公式**
本节主要推导了利用边界值重构区域内部函数值的积分公式：
*   **上半平面的 Poisson 公式**：根据实轴上的数值（或其虚、实部）决定上半平面内任意点的函数值。
*   **圆内区域内的 Poisson 公式**：根据圆周上的数值决定圆内任意点的函数值及其调和分量。
