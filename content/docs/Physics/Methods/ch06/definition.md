根据您选定的第六章《留数定理及其应用》（对应源文件 `ch_6.pdf`）的内容，本章中定义和使用的主要符号及其数学/几何含义总结如下：

### **1. 留数定理与基本概念**
*   **\\( G \\)**：复平面内的**有界区域**。
*   **\\( C \\)**：区域 \\( G \\) 的边界，为**分段光滑的简单闭合曲线**。
*   **\\( b_k \\)（或 \\( z_0 \\)）**：函数 \\( f(z) \\) 在区域 \\( G \\) 内的**孤立奇点**。
*   **\\( f(z) \\)**：除有限个孤立奇点外在 \\( G \\) 内单值解析、在闭区域 \\( \bar{G} \\) 及边界 \\( C \\) 上连续的**复变函数**。
*   **\\( \text{res } f(b_k) \\)**：函数 \\( f(z) \\) 在孤立奇点 \\( b_k \\) 处的**留数** (Residue)。
*   **\\( a_{-1}^{(k)} \\)（或 \\( a_{-1} \\)）**：函数 \\( f(z) \\) 在奇点 \\( b_k \\) 处的 Laurent 展开式中 **\\( (z-b_k)^{-1} \\) 项的系数**。
*   **\\( m \\)**：极点 \\( b \\) 的**阶数** (Order)。
*   **\\( P(z), Q(z) \\)**：分式型函数 \\( P(z)/Q(z) \\) 的分子与分母，用于简化一阶及高阶极点的留数计算。

### **2. 无穷远点留数**
*   **\\( \text{res } f(\infty) \\)**：函数 \\( f(z) \\) 在**无穷远点 \\( \infty \\) 处的留数**。
*   **\\( C' \\)**：绕无穷远点 \\( \infty \\) **顺时针方向（即绕 \\( \infty \\) 点的正方向）**一周的简单封闭曲线。
*   **\\( t \\)**：变换变量 \\( t = 1/z \\)，用于将无穷远点 \\( z = \infty \\) 的行为映射到原点 \\( t = 0 \\) 进行研究。

### **3. 实积分计算与围道几何元素**
*   **\\( R(\sin \theta, \cos \theta) \\)**：有理三角函数积分中的**有理被积函数**。
*   **\\( z = e^{i\theta} \\)**：计算有理三角函数积分时引入的**复变量代换**。
*   **\\( V.p. \int \\)**：**积分主值** (Cauchy Principal Value)，在无穷积分或积分路径上有奇点（瑕积分）时使用。
*   **\\( C_R \\)**：以原点为圆心、\\( R \\) 为半径的**上半圆弧（大圆弧）**，常用于构造闭合积分围道。
*   **\\( [-R, R] \\)**：实轴上的积分线段，与 \\( C_R \\) 共同构成闭合围道。
*   **\\( C_\delta \\)（或 \\( C_s \\)）**：以奇点为圆心、\\( \delta \\)（或 \\( s \\)）为半径的**小半圆弧**，用于在积分路径上绕开奇点。
*   **\\( z^{s-1} Q(z) \\)**：涉及多值函数（如分支点 \\( z=0 \\) 和 \\( \infty \\)）积分时的典型复变被积函数，其中 **\\( s \\) 为实数**。

### **4. 无穷级数求和**
*   **\\( C_N \\)**：用于计算某些无穷级数之和的**正方形积分围道**，其四个顶点位于 \\( (N + 1/2)(1 \pm i) \\) 处。
*   **\\( \pi \cot \pi z \\)**：级数求和中引入的**特殊辅助函数**，在所有整数点 \\( z=n \\) 处有一阶极点且留数均为 \\( 1 \\)。
