### **§ 6.1 留数定理**
本节介绍了留数的定义、留数定理以及在复平面有限孤立奇点处的留数计算方法。

*   **定理 6.1 (留数定理)**：设有界区域 \\( G \\) 的边界 \\( C \\) 为分段光滑的简单闭合曲线。若除有限个孤立奇点 \\( b_k \\)（\\( k = 1, 2, 3, \dots, n \\)）外，函数 \\( f(z) \\) 在 \\( G \\) 内单值解析，在 \\( \bar{G} \\) 中连续，并且 \\( f(z) \\) 在边界 \\( C \\) 上连续，则沿区域 \\( G \\) 边界正向的积分满足：
    \\( \oint_C f(z)dz = 2\pi i \sum_{k=1}^n \text{res } f(b_k) \\)
*   **\\( m \\) 阶极点的留数计算定理（公式）**：设 \\( z=b \\) 是 \\( f(z) \\) 的 \\( m \\) 阶极点，则其留数为：
    \\( \text{res } f(b) = \frac{1}{(m-1)!} \lim_{z \to b} \frac{d^{m-1}}{dz^{m-1}} \left[ (z-b)^m f(z) \right] \\)
*   **一阶极点的留数计算定理（公式）**：设 \\( z=b \\) 是 \\( f(z) \\) 的一阶极点，则其留数为：
    \\( \text{res } f(b) = \lim_{z \to b} (z-b)f(z) \\)
*   **分式型一阶极点留数定理（公式）**：若 \\( f(z) = \frac{P(z)}{Q(z)} \\)，且 \\( P(b) \neq 0 \\)，\\( z=b \\) 是 \\( Q(z) \\) 的一阶零点（即 \\( Q(b) = 0, Q'(b) \neq 0 \\)），则其留数为：
    \\( \text{res } f(b) = \frac{P(b)}{Q'(b)} \\)

### **§ 6.2 有理三角函数的积分**
*   *本节主要介绍通过引入变换 \\( z = e^{i\theta} \\) 将区间 \\( [0, 2\pi] \\) 上的有理三角函数积分转化为单位圆周 \\( |z|=1 \\) 上的围道积分的方法，**未包含**独立命名的定理或引理。*
*   **边界有一阶极点时的三角函数积分结论（练习 6.7）**：若 \\( f(z) \\) 在单位圆周 \\( |z|=1 \\) 上有有限个一阶极点 \\( \rho_k \\)（\\( k=1,2,\dots,m \\)），则：
    \\( \int_0^{2\pi} R(\sin\theta, \cos\theta) d\theta = 2\pi \sum_{|z|<1} \text{res } \frac{f(z)}{z} + \pi \sum_{k=1}^m \text{res } \frac{f(z)}{z}_{z=\rho_k} \\)

### **§ 6.3 无穷积分**
本节讨论利用留数定理计算实轴无穷积分 \\( \int_{-\infty}^{\infty} f(x)dx \\) 的方法。

*   **大圆弧积分趋于零的条件（收敛判定）**：若函数 \\( f(z) \\) 在上半平面除有限个孤立奇点外处处解析，且在实轴上没有奇点；当 \\( 0 \le \arg z \le \pi \\) 且 \\( |z| \to \infty \\) 时，\\( zf(z) \\) 一致地趋于 \\( 0 \\)，则沿上半圆弧 \\( C_R \\) 的积分极限为零，即：
    \\( \lim_{R \to \infty} \int_{C_R} f(z) dz = 0 \\)
*   **无穷多个孤立奇点时的级数求和定理（公式 6.19）**：若在上半平面有无穷多个孤立奇点 \\( b_n \\)（\\( n = 1, 2, \dots \\)），且存在圆弧序列 \\( \{C_m\} \\) 满足特定条件使得沿其积分极限为零，则：
    \\( \int_{-\infty}^{\infty} f(x)dx = 2\pi i \sum_{n} \text{res } f(b_n) \\)

### **§ 6.4 含三角函数的无穷积分**
本节重点给出了克服三角函数在无穷远点具有本性奇点这一困难的重要引理。

*   **引理 6.1 (Jordan 引理)**：设在 \\( 0 \le \arg z \le \pi \\) 范围内，当 \\( |z| \to \infty \\) 时 \\( Q(z) \\) 一致地趋于 \\( 0 \\)，则对于 \\( p > 0 \\)：
    \\( \lim_{R \to \infty} \int_{C_R} Q(z) e^{ipz} dz = 0 \\)
    其中 \\( C_R \\) 是以原点为圆心、\\( R \\) 为半径的上半圆弧。
*   **补充引理一**：设函数 \\( Q(z) \\) 只有有限个奇点，且在下半平面的范围内，当 \\( |z| \to \infty \\) 时一致地趋近于 \\( 0 \\)，则对于 \\( p > 0 \\)：
    \\( \lim_{R \to \infty} \int_{C_R} Q(z)e^{-ipz}dz = 2\pi i \sum_{\text{下半平面}} \text{res } \{ Q(z)e^{-ipz} \} \\)
    其中 \\( C_R \\) 是位于上半平面的半圆弧。
*   **补充引理二**：在与上述引理相同的条件下，大圆弧积分可以用无穷远点的留数表示为：
    \\( \lim_{R \to \infty} \int_{C_R} Q(z)e^{-ipz}dz = -2\pi i \cdot \text{res } \{ Q(z)e^{-ipz} \}_{z=\infty} \\)

### **§ 6.5 积分路径上有奇点的情形**
本节主要探讨积分路径上含有一阶极点（瑕点）时的实变积分计算。

*   **小半圆弧积分极限性质（基于引理 3.4）**：当积分路径上存在一阶极点 \\( z = a \\) 时，围绕其作半径为 \\( \delta \\) 的小半圆弧 \\( C_\delta \\)。当 \\( \delta \to 0 \\) 时，沿该小圆弧的积分极限为：
    \\( \lim_{\delta \to 0} \int_{C_\delta} f(z) dz = \pm i \pi \cdot \text{res } f(a) \\)
    （符号由绕行方向决定，逆时针取正，顺时针取负）。

### **§ 6.8 应用留数定理计算无穷级数的和**
本节介绍了利用留数定理将无穷级数求和问题转化为复变函数留数计算的理论与引理。

*   **引理 6.2**：设 \\( f(z) \\) 在 \\( \mathbb{C} \\) 内除了有限个孤立奇点外处处解析。若存在常数 \\( R>0 \\) 和 \\( M>0 \\)，使当 \\( |z| > R \\) 时，\\( |z f(z)| \le M \\)，则有：
    \\( \lim_{N \to \infty} \oint_{C_N} \pi \cot \pi z f(z) dz = 0 \\)
    其中积分围道 \\( C_N \\) 是以原点为中心、四个顶点位于 \\( (N + \frac{1}{2})(1 \pm i) \\) 和 \\( -(N + \frac{1}{2})(1 \pm i) \\) 处的正方形围道。
*   **定理 6.2**：若函数 \\( f(z) \\) 在 \\( \mathbb{C} \\) 上除有限个非整数的极点外处处解析，且存在常数 \\( R>0 \\) 和 \\( M>0 \\)，使当 \\( |z| > R \\) 时，\\( |z f(z)| \le M \\)，则级数求和公式为：
    \\( \sum_{n=-\infty}^{\infty} f(n) = -\sum_{f(z) \text{ 的极点}} \text{res } \{ \pi \cot \pi z f(z) \} \\)
