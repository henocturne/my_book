根据源文件《第七章 \\( \Gamma \\)

### 1. \\( \Gamma \\)函数与函数基础符号
*   **\\( \Gamma(z) \\)
**：**\\( \Gamma \\)
 函数**（阶乘函数），定义为第二类 Euler 积分：  
    \\( \Gamma(z) = \int_0^\infty e^{-t} t^{z-1} dt \quad (\text{Re } z > 0) \quad \\)

*   **\\( z \\)
**：复平面 \\( \mathbb{C} \\)
 上的**复自变量**。
*   **\\( t \\)
**：第二类 Euler 积分中的**积分变量**，通常满足 \\( \arg t = 0 \\)
。在修改路径时，也可延拓为复变量。
*   **\\( \Gamma(z, a) \\)
 和 \\( \gamma(z, a) \\)
**：**不完全 \\( \Gamma \\)
 函数**，定义分别为 \\( \Gamma(z, a) = \int_a^\infty e^{-t} t^{z-1} dt \\)
 和 \\( \gamma(z, a) = \int_0^a e^{-t} t^{z-1} dt \\)
（脚注）。
*   **\\( L \\)
**：\\( t \\)
 复平面内从 \\( t=0 \\)
 出发的**积分路径**（半射线或分段光滑曲线），满足 \\( \arg t = \alpha \\)
 且 \\( |\alpha| < \pi/2 \\)
。
*   **\\( \text{res }\Gamma(-n) \\)
**：\\( \Gamma \\)
 函数在负整数和零点 \\( z = -n \\)
（\\( n = 0, 1, 2, \dots \\)
 一阶极点）处的**留数**。
*   **\\( B(p, q) \\)
**：**\\( B \\)
 函数**（Beta 函数），由第一类 Euler 积分定义，用于复平面上的解析延拓。
*   **\\( p, q \\)
**：\\( B \\)
 函数的**复自变量**。

---

### **2. \\( \psi \\)函数及常数符号**
*   **\\( \psi(z) \\)
**：**\\( \psi \\)
 函数**（双伽马函数），定义为 \\( \Gamma \\)
 函数的对数微商：  
    \\( \psi(z) = \frac{d \ln \Gamma(z)}{dz} = \frac{\Gamma'(z)}{\Gamma(z)} \quad \\)

*   **\\( \gamma \\)
**：**Euler 常数**（或称 Euler-Mascheroni 常数），是最基本的数学常数之一，满足 \\( \gamma = -\psi(1) \approx 0.577215... \\)
。

---

### **3. 级数求和与代数分析符号**
*   **\\( u_n \\)
**：通项为有理式的**无穷级数项**。
*   **\\( p(n), d(n) \\)
**：有理通项级数 \\( u_n \\)
 的**分子与分母多项式**。
*   **\\( \alpha_k, a_k \\)
**：有理式 \\( u_n \\)
 在部分分式展开时的**极点与展开系数**。
*   **\\( C_N \\)
**：求极点或有理分式展开时引入的**闭合围道**。

---

### **4. 物理与统计应用相关符号**
*   **\\( v \\)
**：Maxwell 速率分布中的**粒子速率**。
*   **\\( \langle v^n \rangle \\)
**：速率 \\( v^n \\)
 的**平均值**。
*   **\\( m, k, T \\)
**：分别为**粒子质量、Boltzmann 常数及热力学温度**。
*   **\\( f(x) \\)
**：\\( \Gamma \\)
 分布的**概率密度函数**。
*   **\\( \alpha, \beta \\)
**：\\( \Gamma \\)
 分布中已知的**正数参数**。
*   **\\( \langle x \rangle \\)
 和 \\( \sqrt{\langle x^2 \rangle - \langle x \rangle^2} \\)
**：\\( \Gamma \\)
 分布的**平均值与标准偏差**。
