### **§ 7.1 \\( \Gamma \\) 函数的定义**
本节主要给出 \\( \Gamma \\) 函数的积分定义，并证明其在复平面上的解析性及延拓性质：

*   **右半平面解析性结论**：通过将第二类 Euler 积分拆分为两部分讨论（\\( \Gamma(z) = \int_0^1 e^{-t}t^{z-1}dt + \int_1^\infty e^{-t}t^{z-1}dt \\)），证明了其第一部分的一致收敛性及第二部分在全复平面的解析性，从而确立了：**\\( \Gamma(z) \\) 函数在复半平面 \\( \text{Re } z > 0 \\) 内是一个单值解析函数**。
*   **解析延拓结论**：通过将指数函数 \\( e^{-t} \\) 进行 Taylor 展开并进行逐项积分，得到了 \\( \Gamma(z) \\) 在全复平面（除非正整数点外）的解析延拓级数表示式：
    \\( \Gamma (z) = \int_1^\infty e^{-t}t^{z-1}dt + \sum_{n=0}^\infty \frac{(-1)^n}{n!(n+z)} \quad (z \neq 0, -1, -2, \dots) \\)
    由此证明该级数在全复平面一致收敛，即实现了将 \\( \Gamma(z) \\) 从右半平面向全复平面的解析延拓。

### **§ 7.2 \\( \Gamma \\) 函数的基本性质**
本节给出了 \\( \Gamma \\) 函数的最核心性质与推论：

*   **性质 2（递推关系定理）**：\\( \Gamma(z+1) = z\Gamma(z) \\)。该关系不仅在右半平面成立，根据解析延拓原理，它在全复平面（非正整数除外）均成立。
*   **极点与留数结论**：由递推关系可将 \\( \Gamma(z) \\) 延拓至全复平面，其在 \\( z = 0, -1, -2, \dots \\) 处均为一阶极点，在极点处的留数为：
    \\( \text{res }\Gamma (-n) = \frac{(-1)^n}{n!} \\)
*   **推论 7.1**：对于正整数 \\( n \\)，有 \\( \Gamma(n) = (n-1)! \\)。
*   **性质 3（互余宗量关系 / 反射公式）**：
    \\( \Gamma (z)\Gamma (1-z) = \frac{\pi}{\sin \pi z} \\)
    该公式在全复平面均成立。
*   **推论 7.2**：\\( \Gamma (1/2) = \sqrt{\pi} \\)。
*   **推论 7.3**：\\( \Gamma \\) 函数在全复平面上无零点。
*   **性质 4（倍乘公式）**：
    \\( \Gamma (2z) = \frac{2^{2z-1}}{\sqrt{\pi}} \Gamma (z)\Gamma \left(z+\frac{1}{2}\right) \\)
    该公式在全复平面（除极点外）均成立。
*   **性质 5（Stirling 渐近展开公式）**：当 \\( |z| \to \infty \\) 且 \\( |\arg z| < \pi \\) 时，有：
    \\( \Gamma (z) \sim \sqrt{2\pi} z^{z-\frac{1}{2}} e^{-z} \left[ 1 + \frac{1}{12z} + \frac{1}{288z^2} + \dots \right] \\)
    并由此在物理中导出常用的近似对数式 \\( \ln n! \sim n\ln n - n \\)。

### **§ 7.3 \\( \psi \\) 函数**
本节讨论 \\( \Gamma \\) 函数对数微商的性质，并给出了无穷级数求和的重要工具：

*   **\\( \psi \\) 函数极点性质**：\\( z=0, -1, -2, \dots \\) 都是 \\( \psi(z) \\) 的一阶极点，留数均为 \\( -1 \\)；除这些点外，\\( \psi(z) \\) 在全复平面解析。
*   **一阶极点有理级数求和定理（公式 7.29）**：若级数通项 \\( u_n = \frac{p(n)}{d(n)} \\) 只有一阶极点且满足收敛条件 \\( \sum_{k=1}^m a_k = 0 \\)，则级数和可以通过 \\( \psi \\) 函数计算：
    \\( \sum_{n=0}^\infty u_n = -\sum_{k=1}^m a_k \psi (\alpha_k) \\)
*   **二阶极点有理级数求和定理（公式 7.31c）**：若级数通项包含二阶极点且满足收敛条件 \\( \sum a_k + \sum b_{1k} = 0 \\)，则其求和公式为：
    \\( \sum_{n=0}^\infty u_n = -\sum_{k=1}^m a_k \psi(\alpha_k) - \sum_{k=1}^l \left[ b_{1k}\psi(\beta_k) - b_{2k}\psi'(\beta_k) \right] \\)

### **§ 7.4 \\( B \\) 函数**
本节建立了 \\( B \\) 函数（Beta 函数）与 \\( \Gamma \\) 函数的定量联系：

*   **\\( B \\) 函数与 \\( \Gamma \\) 函数转换关系定理（核心公式 7.34）**：若 \\( \text{Re } p > 0, \text{Re } q > 0 \\)，则：
    \\( B(p, q) = \frac{\Gamma (p)\Gamma (q)}{\Gamma (p+q)} \\)
    利用该关系式，可将双变量 \\( B(p, q) \\) 函数解析延拓到 \\( p \\) 或 \\( q \\) 的全复平面。
*   **对称性结论**：\\( B(p, q) = B(q, p) \\)。

### **§ 7.5 \\( \Gamma \\) 函数的普遍表达式**
本节不加证明地给出了在全复平面上（孤立奇点除外）成立的三种普遍表达式：

*   **围道积分表示定理（一）**（绕原点正向一周的路径）：
    \\( \Gamma (z) = -\frac{1}{2i\sin\pi z} \int_{C(0+)} e^{-t}(-t)^{z-1}dt \quad (|\arg(-t)| < \pi) \\)
*   **围道积分表示定理（二）**（全复平面均成立，含整数点）：
    \\( \frac{1}{\Gamma (z)} = \frac{1}{2\pi i} \int_{C} e^t t^{-z}dt \quad (|\arg t| < \pi) \\)
*   **Euler 无穷乘积表示定理**：
    \\( \Gamma(z)=\frac{1}{z}\prod_{n=1}^\infty \left[ \left(1+\frac{1}{n}\right)^z \left(1+\frac{z}{n}\right)^{-1} \right] \\)
*   **Weierstrass 无穷乘积表示定理**：
    \\( \frac{1}{\Gamma (z)} = z e^{\gamma z} \prod_{n=1}^\infty \left[ \left(1+\frac{z}{n}\right)e^{-z/n} \right] \\)
*   **有理分式展开性质**：从上述无穷乘积表示中，可以导出三角函数（如 \\( \tan \pi z, \cot \pi z, \csc \pi z \\)）的**有理分式展开式**。此类展开形式不同于 Taylor 或 Laurent 展开，其收敛范围是除孤立奇点外的全复平面，能够将亚纯函数在所有极点处的奇异性同时表现出来。

### **§ 7.6 \\( \Gamma \\) 函数的渐近展开**
*   **Stirling 公式实数推导结论**：利用鞍点法的基本思想，在实数范围 \\( x > 0 \\) 内，通过分析被积函数 \\( e^{-t}t^x \\) 在极大值点 \\( t=x \\) 附近的局域性质进行 Taylor 展开，推导并验证了级数前两阶待定系数为 \\( A = 1/12, B = 1/288 \\) 的 Stirling 渐近展开结构。
