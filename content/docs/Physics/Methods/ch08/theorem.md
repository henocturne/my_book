### **§ 8.1 Laplace 变换的定义**

本节主要给出拉氏变换的数学定义，并确立了变换存在性的核心判定依据：

*   **Laplace 变换存在的充分条件（定理性质）**  ：
    若函数 \\( f(t) \\)  在正实轴上满足以下两个条件，则其 Laplace 变换一定存在：
    1.  **分段连续性**：\\( f(t) \\)  在区间 \\( 0 \le t < \infty \\)  上除第一类间断点外都是连续的，且有连续导数，在任何有限区间内这种间断点的数目是有限的  ；
    2.  **有限增长性**：\\( f(t) \\)  具有有限的增长指数，即存在正数 \\( M > 0, t_0 > 0 \\)  及 \\( \beta' \ge 0 \\) ，使对于任何 \\( t > t_0 \\) ，恒有：
        \\( |f(t)| < M e^{\beta' t} \quad (8.7) \quad \text{ } \\) 

---

### **§ 8.2 Laplace 变换的基本性质**

本节阐述了像函数与原函数在复数域运算中的五大基本性质，这些性质在数学上均具有定理的地位：

*   **性质 1（线性变换定理）**  ：
    Laplace 变换是一个线性变换。对任意复常数 \\( \alpha_1 \\)  和 \\( \alpha_2 \\) ，恒有：
    \\( \mathcal{L} \{\alpha_1 f_1(t) + \alpha_2 f_2(t)\} = \alpha_1 F_1(p) + \alpha_2 F_2(p) \quad (8.8) \quad \text{ } \\) 
*   **性质 2（像函数的解析性定理）**  ：
    若函数 \\( f(t) \\)  满足 Laplace 变换存在的充分条件，则其像函数 \\( F(p) \\)  在半平面 \\( \text{Re } p > s_0 \\) （\\( s_0 \\)  为收敛横标）内是一个**解析函数**  。
*   **性质 3（像函数无穷远处极限性质）**  ：
    若 \\( f(t) \\)  满足 Laplace 变换存在的充分条件，则当 \\( \text{Re } p = s \to +\infty \\)  时，其像函数极限为零，即：
    \\( \lim_{\text{Re } p \to +\infty} F(p) = 0 \quad (8.11) \quad \text{ } \\) 
    此外，根据 **Riemann-Lebesgue 定理**，当 \\( \text{Re } p = s > s_0 \\)  时，亦满足  ：
    \\( \lim_{\text{Im } p \to \infty} F(p) = 0 \quad \text{ } \\) 
*   **性质 4（原函数导数的 Laplace 变换定理）**  ：
    设 \\( f(t) \\)  及其各阶导数的 Laplace 变换均存在，且 \\( f(t) \risingdotseq F(p) \\) ，则导数的变换自动包含初值项：
    *   **一阶导数**：\\( \mathcal{L} \{f'(t)\} = p F(p) - f(0) \quad (8.12) \quad \text{ } \\) 
    *   **\\( n \\)  阶导数**：\\( \mathcal{L} \{f^{(n)}(t)\} = p^n F(p) - p^{n-1} f(0) - p^{n-2} f'(0) - \dots - f^{(n-1)}(0) \quad (8.13\text{c}) \quad \text{ } \\) 
*   **性质 5（原函数积分的 Laplace 变换定理）**  ：
    设 \\( f(t) \\)  满足 Laplace 变换存在的充分条件，其积分的 Laplace 变换为：
    \\( \mathcal{L}   \int_0^t f(\tau) d\tau   = \frac{F(p)}{p} \quad (8.18) \quad \text{ } \\)

### **§ 8.3 Laplace 变换的反演**

本节探讨了如何由像函数还原为原函数，并给出了著名的卷积定理：

*   **反演的唯一性定理**  ：
    如果限定原函数为**连续函数**，则 Laplace 变换的反演具有**唯一性**  。若允许原函数不连续，对应于同一个像函数，其原函数可相差一个几乎处处为零的“零函数”  。
*   **像函数导数的反演定理**  ：
    设 \\( f(t) \\)  满足 Laplace 变换存在的充分条件，且 \\( f(t) \risingdotseq F(p) \\) ，则：
    \\( F^{(n)}(p) \risingdotseq (-t)^n f(t) \quad (8.23) \quad \text{ } \\) 
*   **像函数积分的反演定理**  ：
    若像函数的积分存在，则其反演公式为：
    \\( \mathcal{L}^{-1}   \int_p^\infty F(q) dq   = \frac{f(t)}{t} \quad (8.27) \quad \text{ } \\) 
*   **定理 8.1（卷积定理）**  ：
    设 \\( F_1(p) \risingdotseq f_1(t) \\)  且 \\( F_2(p) \risingdotseq f_2(t) \\) ，则两个像函数乘积的反演等于对应原函数的**卷积**：
    \\( \mathcal{L}\{f_1(t) * f_2(t)\} = F_1(p) F_2(p) \quad (8.36) \quad \text{ } \\) 
    其中卷积定义为：\\( f_1(t) * f_2(t) = \int_0^t f_1(\tau) f_2(t-\tau) d\tau \\)   。

### **§ 8.4 普遍反演公式**

本节将反演推广到复平面上的复变积分，并给出了计算所需的重要引理：

*   **普遍反演定理（Mellin 反演公式）**  ：
    若复变函数 \\( F(p) \\)  在区域 \\( \text{Re } p > s_0 \\)  内满足：
    1.  \\( F(p) \\)  解析  ；
    2.  当 \\( |p| \to \infty \\)  时 \\( F(p) \\)  趋于 \\( 0 \\)   ；
    3.  沿直线 \\( L: \text{Re } p = s > s_0 \\)  的无穷积分 \\( \int_{-\infty}^{\infty} |F(s+i\sigma)| d\sigma \\)  收敛  ；
    则 \\( F(p) \\)  的原函数可通过复平面上的围道积分（**Mellin 反演积分**）确定：
    \\( f(t) = \frac{1}{2\pi i} \int_{s-i\infty}^{s+i\infty} F(p) e^{pt} dp \quad (s > s_0) \quad (8.38) \quad \text{ } \\) 
*   **推广的 Jordan 引理**  ：
    为了在复反演积分中估算大圆弧的积分极限，将原始的 Jordan 引理作了如下扩充：
    将 Jordan 引理所讨论的圆弧**旋转了 \\( 90^\circ \\) **  ，使半圆弧与直线 \\( \text{Re } p = s > 0 \\)  相交（略大于半圆弧）  。在圆弧与虚轴的距离 \\( s \\)  固定时，当半径 \\( R \to \infty \\) ，虚轴右方圆弧的张角趋于 \\( 0 \\)   。在此条件下，若代数项在大圆弧上一致趋于 0，则沿该大圆弧的积分极限为 0  。
*   **引理 3.4（在多值函数反演中的应用）**  ：
    在计算含分支点（如 \\( p=0 \\) ）的多值函数反演时，围绕分支点的极小半圆弧 \\( C_\delta \\)  的积分，当半径 \\( \delta \to 0 \\)  时极限为零  ：
    \\( \lim_{\delta \to 0} \int_{C_\delta} \dots dp = 0 \quad \text{ } \\) 

---

### **§ 8.5 利用 Laplace 变换计算级数和**

*   **级数和与积分转换定理**  ：
    利用 Laplace 变换，可以通过交换积分与级数求和的次序，将离散的无穷级数求和问题转化为定积分计算：
    \\( \sum_{n=1}^\infty F(n) = \int_0^\infty f(t) \left( \sum_{n=1}^\infty e^{-nt} \right) dt = \int_0^\infty \frac{f(t)}{e^t-1} dt \quad (8.45) \quad \text{ } \\) 
    （由于指数项 \\( e^{-nt} \\)  的存在，在一般情况下常能保证交换积分与求和次序的合法性  ）。
