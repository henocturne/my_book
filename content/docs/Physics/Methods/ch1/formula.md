### **一、 主要内容总结**
本章主要介绍了复数的基础理论及其在几何和函数层面的扩展。
1.  **复数的代数运算与几何结构**：定义了复数作为有序实数对的运算规则，引入了虚单位 \\( i \\) 及标准代数形式 \\( z = x + iy \\)。
2.  **复数的几何表示**：阐述了复平面、共轭复数、模与辐角的概念，并介绍了复数的极坐标与指数表示法（Euler 公式）。
3.  **复数序列**：探讨了复数序列的聚点、上/下极限、有界性以及收敛性的 Cauchy 充要条件。
4.  **复变函数与区域**：定义了复平面上的点集、区域（单连通与多连通）、边界，以及复变函数 \\( w = f(z) \\) 的映射本质。
5.  **扩充复平面**：引入了无穷远点 \\( \infty \\) 和 Riemann 球面的概念。
6.  **几何应用**：以正十七边形的尺规作图问题为例，展示了复数在几何证明中的应用。

### **二、 定义整理（按指定格式）**

*   **定义 1.1.1（复数）**：设 \\( x \\) 和 \\( y \\) 都是实数，有序实数对 \\( (x, y) \\) 遵从特定的加法与乘法规则，则称其定义了一个复数 \\( z \\)。
*   **定义 1.1.2（实部与虚部）**：在复数 \\( z = (x, y) \\) 中，\\( x \\) 称为实部（\\( \text{Re } z \\)），\\( y \\) 称为虚部（\\( \text{Im } z \\)）。
*   **定义 1.1.3（虚单位）**：复数 \\( (0, 1) \\) 称为虚单位，记作 \\( i \\)，满足 \\( i^2 = -1 \\)。
*   **定义 1.1.4（共轭复数）**：复数 \\( z^* = x - iy \\) 与 \\( z = x + iy \\) 互为复共轭。
*   **定义 1.1.5（模与辐角）**：\\( r = |z| \\) 称为复数的模，\\( \theta = \arg z \\) 称为复数的辐角。
*   **定义 1.2.1（复数序列）**：按照一定顺序排列的无穷个复数 \\( \{z_n\} \\) 称为复数序列。
*   **定义 1.2.2（聚点/极限点）**：若存在复数 \\( z \\)，对于任意 \\( \epsilon > 0 \\)，恒有无穷多个 \\( n \\) 使得 \\( |z_n - z| < \epsilon \\)，则称 \\( z \\) 为序列的一个聚点。
*   **定义 1.2.3（收敛与极限）**：若对于任意 \\( \epsilon > 0 \\)，存在正整数 \\( N(\epsilon) \\)，当 \\( n > N \\) 时恒有 \\( |z_n - z| < \epsilon \\)，则称序列收敛于 \\( z \\)。
*   **定义 1.3.1（内点）**：如果以该点为圆心能作一个圆，使圆内所有点都属于点集，则该点称为点集的内点。
*   **定义 1.3.2（区域）**：全部由内点组成且具有连通性的点集称为区域。
*   **定义 1.3.3（边界点与边界）**：若点 \\( z_1 \\) 的任意邻域内总含有区域 \\( G \\) 的点且 \\( z_1 \notin G \\)，则称其为边界点。边界点的全体构成边界 \\( C \\)。
*   **定义 1.3.4（复变函数）**：对于区域 \\( G \\) 内的每一个复数 \\( z \\)，都有唯一一个复数 \\( w \\) 与之对应，则称 \\( f \\) 为定义在 \\( G \\) 上的复变函数。
*   **定义 1.4.1（无穷远点）**：无界序列会聚的特殊点，记为 \\( \infty \\)，其模大于任何正数，辐角不定。

### **三、 公式整理（按原文序号）**

*   **(1.1)** 加法规则：\\( (x_1, y_1) + (x_2, y_2) = (x_1 + x_2, y_1 + y_2) \\)
*   **(1.2)** 乘法规则：\\( (x_1, y_1)(x_2, y_2) = (x_1x_2 - y_1y_2, x_1y_2 + y_1x_2) \\)
*   **(1.3)** 复数记法：\\( z = (x, y) = x(1, 0) + y(0, 1) \\)
*   **(1.4)** 虚单位定义：\\( i \equiv (0, 1) \\)
*   **(1.5)** 代数形式：\\( z = x + iy \\)
*   **(1.6)** 减法规则：\\( z_1 - z_2 = (x_1 - x_2, y_1 - y_2) \\)
*   **(1.7)** 除法运算：\\( \frac{x_1 + iy_1}{x_2 + iy_2} = \frac{x_1x_2 + y_1y_2}{x_2^2 + y_2^2} + i \frac{y_1x_2 - x_1y_2}{x_2^2 + y_2^2} \\)
*   **(1.8)** 极坐标形式：\\( z = r(\cos \theta + i \sin \theta) \\)
*   **(1.9)** 模与辐角定义：\\( r = |z| = \sqrt{x^2+y^2}, \theta = \arg z \\)
*   **(1.10a)** 模相等：\\( |z_1| = |z_2| \\)
*   **(1.10b)** 辐角关系：\\( \arg z_1 = \arg z_2 + 2n\pi \\)
*   **(1.11)** 极坐标乘法：\\( z_1 z_2 = r_1 r_2 [\cos(\theta_1 + \theta_2) + i \sin(\theta_1 + \theta_2)] \\)
*   **(1.12)** 极坐标除法：\\( \frac{z_1}{z_2} = \frac{r_1}{r_2} [\cos(\theta_1 - \theta_2) + i \sin(\theta_1 - \theta_2)] \\)
*   **(1.13)** Euler 公式相关性质：\\( e^{i\theta_1} \cdot e^{i\theta_2} = e^{i(\theta_1 + \theta_2)} \\)
*   **(1.14)** 指数倒数：\\( (e^{i\theta})^{-1} = e^{-i\theta} \\)
*   **(1.15)** 指数形式：\\( z = re^{i\theta} \\)
*   **(1.11')** 指数形式乘法：\\( z_1 \cdot z_2 = r_1 r_2 e^{i(\theta_1 + \theta_2)} \\)
*   **(1.12')** 指数形式除法：\\( \frac{z_1}{z_2} = \frac{r_1}{r_2} e^{i(\theta_1 - \theta_2)} \\)
*   **(1.16)** 序列聚点示例：\\( \{z_n\} = \frac{n^2+(-1)^n n}{n^2} \sin \frac{n\pi}{2} \\)
*   **(1.17a)** 上极限性质：\\( \varlimsup_{n \to \infty} (x_n y_n) \le \varlimsup_{n \to \infty} x_n \cdot \varlimsup_{n \to \infty} y_n \\)
*   **(1.17b)** 下极限性质：\\( \varliminf_{n \to \infty} (x_n y_n) \ge \varliminf_{n \to \infty} x_n \cdot \varliminf_{n \to \infty} y_n \\)
