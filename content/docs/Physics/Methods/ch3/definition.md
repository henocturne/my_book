根据源文件《第三章 复变积分》，本章中定义和使用的主要符号及其含义总结如下：

### **1. 复变积分基础定义**
*   **\\( z \\)**：复平面 \\( \mathbb{C} \\) 上的复变量。
*   **\\( C \\)**：复平面内的积分路径（曲线），若闭合则常称为围道。
*   **\\( f(z) \\)**：在曲线 \\( C \\) 上有定义的复变函数。
*   **\\( z_k \\)**：将曲线 \\( C \\) 分割为 \\( n \\) 段时的分点，其中 \\( z_0 = A \\)，\\( z_n = B \\) 为端点。
*   **\\( \zeta_k \\)**：在分段弧 \\( z_{k-1} \to z_k \\) 上任意取的一点。
*   **\\( \Delta z_k \\)**：弧段增量，定义为 \\( z_k - z_{k-1} \\)。
*   **\\( \int_C f(z) dz \\)**：函数 \\( f(z) \\) 沿曲线 \\( C \\) 的复变积分。
*   **\\( u, v \\)**：函数 \\( f(z) \\) 的实部与虚部，即 \\( f(z) = u + iv \\)。
*   **\\( dx, dy \\)**：复微分 \\( dz \\) 的实部与虚部，即 \\( dz = dx + i dy \\)。

### **2. 积分性质与几何量**
*   **\\( C^- \\)**：积分路径 \\( C \\) 的逆向曲线。
*   **\\( M \\)**：函数模 \\( |f(z)| \\) 在曲线 \\( C \\) 上的上界。
*   **\\( l \\)**：积分路径 \\( C \\) 的长度。
*   **\\( \bar{G} \\)**：有界闭区域。
*   **\\( C_0, C_1, \dots, C_n \\)**：多连通区域的边界曲线，通常 \\( C_0 \\) 为外边界，其余为内边界。

### **3. Cauchy 理论相关**
*   **\\( \oint_C f(z) dz \\)**：沿闭合围道 \\( C \\) 的积分，在 Cauchy 定理中其值为 \\( 0 \\)（若满足解析条件）。
*   **\\( F(z) \\)**：函数 \\( f(z) \\) 的**不定积分**，定义为 \\( F(z) = \int_{z_0}^z f(\zeta) d\zeta \\)。
*   **\\( \zeta \\)**：在不定积分或积分公式中常用的辅助积分变量。
*   **\\( f^{(n)}(z) \\)**：解析函数 \\( f(z) \\) 的 **\\( n \\) 阶导数**。

### **4. 极限与路径变形引理**
*   **\\( C_\delta \\)**：以 \\( \alpha \\) 为圆心、\\( \delta \\) 为半径的**小圆弧**（用于小圆弧引理）。
*   **\\( C_R \\)**：以原点为圆心、\\( R \\) 为半径的**大圆弧**（用于大圆弧引理）。
*   **\\( k, K \\)**：在引理中表示当半径趋于 \\( 0 \\) 或 \\( \infty \\) 时，\\( (z-\alpha)f(z) \\) 或 \\( zf(z) \\) 趋向的复常数。

### **5. 特殊函数与映射**
*   **\\( \ln z \\)**：对数函数，在积分中作为 \\( 1/z \\) 的原函数出现（涉及多值性）。
*   **\\( z^\* \\)**：在上半平面 Poisson 公式中，表示点 \\( z \\) 关于实轴的**反演点**（共轭点）。
*   **\\( u(r, \phi), v(r, \phi) \\)**：在极坐标系下表示的调和函数（用于圆内 Poisson 公式）。
