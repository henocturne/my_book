### **一、 主要内容总结**
本章详细讨论了复变函数积分的定义、性质及其在解析函数理论中的核心地位。
1.  **复变积分基础**：定义了复平面上的线积分，并将其转化为两个实变线积分的组合。
2.  **Cauchy 定理**：这是解析函数论的基石，指出解析函数沿闭合围道的积分为零，并由此推导出路径变形定理。
3.  **不定积分与原函数**：证明了在单连通区域内，解析函数的不定积分仍为解析函数，且满足微积分基本定理。
4.  **积分估值引理**：介绍了在计算积分极限时非常有用的**小圆弧引理**和**大圆弧引理**。
5.  **Cauchy 积分公式**：揭示了解析函数最重要的特性，即区域内部的函数值可以由边界上的积分完全确定。
6.  **高阶导数**：证明了解析函数在解析区域内具有各阶导数，并给出了高阶导数的积分表示法。
7.  **含参积分与 Cauchy 型积分**：讨论了 Cauchy 型积分的解析性及其在含参变量积分中的应用。
8.  **Poisson 公式**：针对上半平面和圆内区域，给出了利用边界值确定内部函数实部或虚部的积分公式。

### **二、 定义整理（按 3.x.x 形式整理）**

*   **定义 3.1.1（复变积分）**：设 \\( C \\) 是复平面内一条由 \\( A \\) 点到 \\( B \\) 点的曲线，将 \\( C \\) 任意分割为 \\( n \\) 段，若和数 \\( \sum_{k=1}^n f(\zeta_k)\Delta z_k \\) 的极限存在且与分割方式及点取法无关，则称此极限为函数 \\( f(z) \\) 沿曲线 \\( C \\) 的积分。
*   **定义 3.2.1（不定积分）**：在有界单连通区域 \\( G \\) 内，固定起点 \\( z_0 \\) 而以 \\( z \\) 为变点的积分 \\( F(z) = \int_{z_0}^z f(\zeta) d\zeta \\) 称为 \\( f(z) \\) 的不定积分。
*   **定义 3.2.2（原函数）**：若函数 \\( \Phi(z) \\) 的导数 \\( \Phi'(z) = f(z) \\)，则称 \\( \Phi(z) \\) 为 \\( f(z) \\) 的原函数。
*   **定义 3.6.1（Cauchy 型积分）**：在分段光滑曲线 \\( C \\) 上连续的函数 \\( \phi(\zeta) \\) 所构成的积分 \\( f(z) = \frac{1}{2\pi i} \int_C \frac{\phi(\zeta)}{\zeta - z} d\zeta \\) 称为 Cauchy 型积分。
