    第二章“极限”是数学分析的理论基础，主要探讨了数列极限的严谨定义、基本性质以及实数系的完备性。以下是根据源文件“ch_2_极限.pdf”整理的主要内容总结及带序号的公式。

    第二章主要内容总结#

    1. 数列极限的定义与性质 (§2.1)：

      • 引入了 $\epsilon - N$ 语言来精确刻画数列极限的概念。
      • 阐述了极限的唯一性、夹逼定理（Squeeze Theorem）、有界性（收敛数列必有界） 以及保序性。
      • 介绍了数列极限的四则运算性质以及子列的概念。

    2. 单调数列的极限 (§2.2)：

      • 确立了单调有界原理：单调有界数列必收敛。
      • 通过单调数列定义了自然对数的底 $e$，并给出了相关的估计不等式。
      • 引入了上极限与下极限的概念，用于研究一般有界数列的敛散性。

    3. Cauchy 准则 (§2.3)：

      • 定义了 Cauchy 数列（基本列），并指出数列收敛的充分必要条件是它为 Cauchy 数列。

    4. Stolz 公式 (§2.4)：

      • 提供了处理 $\frac{\infty}{\infty}$ 型和 $\frac{0}{0}$ 型数列极限的工具，被称为数列版的洛必达法则。

    5. 实数系的基本性质 (§2.5)：

      • 系统论述了实数系完备性的几个等价命题：Cantor 闭区间套原理、Heine-Borel 有限覆盖定理、Bolzano-Weierstrass 聚点定理（或称有界数列必有收敛子列） 及其等价性证明。

    原文公式整理#

    根据源文件，带有原文序号的公式整理如下：

    • (2.1) 保序性证明中的 $\epsilon$ 刻画： 针对收敛数列 ${a_n} \to \alpha$ 和 ${b_n} \to \beta$，对给定的 $\epsilon > 0$，存在 $N_1, N_2$ 使得： $$|a_n - \alpha| < \epsilon, \forall n > N_1; \quad |b_n - \beta| < \epsilon, \forall n > N_2$$

    • (2.2) 收敛数列项的定量估计（推论 2.1.6）： 设 $\lim_{n \to \infty} a_n = \alpha$，如果 $\alpha \neq 0$，则存在 $N$，使得当 $n > N$ 时： $$\frac{1}{2}|\alpha| < |a_n| < \frac{3}{2}|\alpha|$$

    • (2.3) 关于自然常数 $e$ 的不等式链： 对于所有 $n \ge 1$，成立以下关系： $$(1+\frac{1}{n})^n < (1+\frac{1}{n+1})^{n+1} < e < (1+\frac{1}{n+1})^{n+2} < (1+\frac{1}{n})^{n+1}$$

    • (2.4) Stolz 公式（第一种情形：分母趋于 $+\infty$）： 设 ${y_n}$ 严格单调增加趋于 $+\infty$，若极限存在，则： $$\lim_{n \to \infty} \frac{x_n - x_{n-1}}{y_n - y_{n-1}} = \alpha \implies \lim_{n \to \infty} \frac{x_n}{y_n} = \alpha$$

    • (2.5) Stolz 公式（第二种情形：$\frac{0}{0}$ 型）： 设 ${y_n}$ 严格单调递减趋于 $0$，且 ${x_n}$ 也收敛到 $0$，若极限存在，则： $$\lim_{n \to \infty} \frac{x_n - x_{n-1}}{y_n - y_{n-1}} = \alpha \implies \lim_{n \to \infty} \frac{x_n}{y_n} = \alpha$$

    根据源文件 ch_2_极限.pdf，第二章各小节下的所有定理、命题、推论及引理整理如下：

    §2.1 数列极限#

    • 命题 2.1.1（唯一性）：如果数列 ${a_n}$ 收敛，则其极限是唯一的。
    • 定理 2.1.2（夹逼定理）：设 ${a_n}, {b_n}, {c_n}$ 均为数列，且 $a_n \le b_n \le c_n$。若 $\lim_{n \to \infty} a_n = \alpha$ 且 $\lim_{n \to \infty} c_n = \alpha$，则 $\lim_{n \to \infty} b_n = \alpha$。
    • 命题 2.1.3（有界性）：收敛数列必是有界数列。
    • 命题 2.1.4（绝对值性质）：设数列 ${a_n}$ 收敛到 $\alpha$，则 ${|a_n|}$ 收敛到 $|\alpha|$。
    • 命题 2.1.5（保序性）：
      1. 如果存在 $N_0$，当 $n > N_0$ 时 $a_n \ge b_n$，则 $\alpha \ge \beta$。
      2. 反之，如果 $\alpha > \beta$，则存在 $N$，使得当 $n > N$ 时 $a_n > b_n$。
    • 推论 2.1.6（保序性的定量估计）：设 $\lim_{n \to \infty} a_n = \alpha$，若 $\alpha \neq 0$，则存在 $N$，使得当 $n > N$ 时，$\frac{1}{2}|\alpha| < |a_n| < \frac{3}{2}|\alpha|$。
    • 命题 2.1.7（极限的四则运算）：收敛数列对于线性组合、乘法、除法（分母不为 0）均保持收敛性且极限满足相应的运算规则。
    • 命题 2.1.8（子列性质）：
      1. 若数列收敛，则其任何子列也收敛到同一极限。
      2. 若数列的偶子列与奇子列均收敛到同一极限，则原数列也收敛。

    §2.2 单调数列的极限#

    • 定理 2.2.1（单调数列极限理论）：
      1. 单调递增数列收敛的充要条件是有上界，其极限为上确界。
      2. 单调递减数列收敛的充要条件是有下界，其极限为下确界。
    • 推论 2.2.2：有界单调数列必收敛。
    • 定理 2.2.3（上、下极限与收敛性）：对于有界数列，收敛的充要条件是其上极限和下极限相等。
    • 命题 2.2.4（上、下极限性质）：包含上、下极限的保序性以及和的上极限不等式。

    §2.3 Cauchy 准则#

    • 命题 2.3.1：Cauchy 数列必为有界数列。
    • 定理 2.3.2（Cauchy 收敛准则）：数列收敛的充分必要条件是它为 Cauchy 数列（基本列）。

    §2.4 Stolz 公式#

    • 引理 2.4.1：关于数列项之和与分母之和比例关系的范围估计。
    • 定理 2.4.2（Stolz 公式之一）：针对分母严格单调增加趋于 $+\infty$ 的情形（$\frac{*}{\infty}$ 型）。
    • 定理 2.4.3（Stolz 公式之二）：针对分母严格单调递减趋于 0 且分子也趋于 0 的情形（$\frac{0}{0}$ 型）。

    §2.5 实数系的基本性质#

    • 定理 2.5.1（Cantor 闭区间套原理）：设 ${[a_n, b_n]}$ 为闭区间套。如果 $\lim_{n \to \infty} (b_n - a_n) = 0$，则这些闭区间有一个唯一的公共点。
    • 定理 2.5.2（Heine-Borel 有限覆盖定理）：设 $\Gamma$ 为 $\mathbb{R}$ 中的一族开集。如果闭区间 $[a, b]$ 包含于这族开集的并集之中，则 $[a, b]$ 必定包含于该族开集的有限个成员的并集之中。
    • 定理 2.5.3（Bolzano-Weierstrass 聚点定理/有界数列必有收敛子列）：有界数列必含有收敛子列。
    • 定理 2.5.4（完备性等价性补充）：证明了 Bolzano 定理与确界原理、Cauchy 收敛准则之间的等价关系。
