    根据您提供的《数学物理方法（第三版）》第一章“复数和复变函数”的内容，按照原文公式序号整理的主要内容如下：

    一、 复数及其运算#

    • (1.1) 复数加法：$(x_{1},y_{1})+(x_{2},y_{2})=(x_{1}+x_{2},y_{1}+y_{2})$。
    • (1.2) 复数乘法：$(x_{1},y_{1})(x_{2},y_{2})=(x_{1}x_{2}-y_{1}y_{2},x_{1}y_{2}+y_{1}x_{2})$。
    • (1.3) 复数定义：$z=(x,y)=x(1,0)+y(0,1)$，其中 $x$ 为实部，$y$ 为虚部。
    • (1.4) 虚单位：$i\equiv(0,1)$，并满足 $i^2 = -1$。
    • (1.5) 代数表示：$z=x+iy$。
    • (1.6) 复数减法：$z_{1}-z_{2}=(x_{1}-x_{2},y_{1}-y_{2})$。
    • (1.7) 复数除法：利用共轭复数将分母实数化：$\frac{x_1 + iy_1}{x_2+iy_2} = \frac{x_1x_2 + y_1y_2 + i(y_1x_2 - x_1y_2)}{x_2^2 + y_2^2}$。

    二、 复数的表示形式#

    • (1.8) 极坐标表示：$z=x+iy=r(\cos\theta+i\sin\theta)$。
    • (1.9) 模与辐角：$r=|z|=\sqrt{x^{2}+y^{2}}$，$\theta=\arg z$。
    • (1.10a, b) 复数相等条件：模相等且辐角相差 $2\pi$ 的整数倍。
    • (1.11) 极坐标乘法：模相乘，辐角相加。
    • (1.12) 极坐标除法：模相除，辐角相减。
    • (1.13) Euler 公式：$e^{i\theta}=\cos\theta+i\sin\theta$。
    • (1.14) 指数函数性质：$e^{i\theta_{1}}\cdot e^{i\theta_{2}}=e^{i(\theta_{1}+\theta_{2})}$。
    • (1.15) 指数表示：$z=re^{i\theta}$。
    • (1.11’) 指数乘法简写：$z_{1}\cdot z_{2}=r_{1}r_{2}e^{i(\theta_{1}+\theta_{2})}$。
    • (1.12’) 指数除法简写：$\frac{z_{1}}{z_{2}}=\frac{r_{1}}{r_{2}}e^{i(\theta_{1}-\theta_{2})}$。

    三、 复数序列与极限#

    • (1.16) 聚点示例：序列 ${(-1)^{n+1}\frac{n}{n+1}}$ 有两个聚点 $\pm 1$。
    • (1.17a, b) 上、下极限不等式：关于非负实数序列乘积的上极限与下极限的性质。

    在第一章中，作者主要通过定义和性质来构建复数体系。虽然文中明确标注为“定理”的内容较少，但涵盖了数个核心的数学准则。

    以下是根据 §1.1 至 §1.5 各小节整理的定理及核心数学性质：

    §1.1 预备知识：复数与复数运算#

    本节侧重于基本定义，并未出现冠以“定理”字样的命题，但确立了复数域的核心运算法则：

    • 复数相等准则：两个复数相等的充分必要条件是它们的实部和虚部分别相等。
    • 乘法交换律与结合律：复数域 $C$ 构成一个域，满足加法和乘法的交换律、结合律及分配律。
    • 几何加法准则：复数加法满足平行四边形法则（或三角形法则）。
    • 复数除法准则：借助共轭复数将分母实数化来完成运算。
    • Euler 公式：$e^{i\theta} = \cos\theta + i\sin\theta$，该公式定义了纯虚数指数函数并保留了实指数函数的性质。

    §1.2 复数序列#

    本节包含了本章最重要的两个形式化定理：

    • Bolzano - Weierstrass 定理：一个有界序列至少有一个聚点。
    • 序列收敛的 Cauchy 充要条件：序列 ${z_n}$ 收敛的充分必要条件是：$\forall\epsilon>0$，$\exists$ 正整数 $N(\epsilon)$，使对于 $\forall$ 正整数 $p$，恒有 $|z_{N+p}-z_N|<\epsilon$。
    • 极限唯一性性质：如果一个序列是收敛的，其极限必然是唯一的聚点。

    §1.3 复变函数#

    本节主要定义了点集、区域及其性质，作为后续微积分讨论的准备：

    • 连通性准则：区域必须满足全部由内点组成且具有连通性（任意两点可用折线连接且折线在集内）。
    • 单连通区域判定：在区域内作任何简单闭合围道，若围道内部的点全部属于该区域，则为单连通区域。

    §1.4 无穷远点#

    • Riemann 球面映射：通过复数球面（Riemann 球面）建立扩充复平面与球面上点之间的一一对应关系，使无穷远点有了直观的几何表示。
    • 无穷远点转换法则：利用映射 $z=1/t$，可以将 $z=\infty$ 的性质研究转化为 $t=0$ 点的性质研究。

    §1.5 正十七边形的尺规作图问题#

    • 代数可作图性：该节证明了正十七边形的边长可以通过有限次加、减、乘、除和开平方运算（平方根）得出，从而证明其可以由圆规和直尺作出。
