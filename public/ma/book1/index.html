    第一章“引言”主要介绍了数学分析中的基本方法和核心概念，重点围绕求和、估计以及实数系的构造展开。以下是基于提供的源文件对第一章主要内容的总结及原文公式的整理。

    第一章主要内容总结#

    1. 从求和谈起（§1.1）：

      • 探讨了自然数序列的求和方法，如高斯的算术级数求和技巧。
      • 引入了裂项相消法，并利用此方法推导了平方和公式。
      • 通过对曲边三角形（如 $y=x^2$ 下方区域）面积的近似求和，初步展示了微积分中“以直代曲”的思想。

    2. 比较与估计（§1.2）：

      • 强调了定量估计是数学分析的核心。通过误差分析（如面积估算中的误差 $E_n$）来确定极限值的存在性。
      • 引入了确界原理（上确界 $sup$ 和下确界 $inf$），这是实数系完备性的基本刻画。
      • 介绍了几类重要的基本不等式，包括三角不等式、均值不等式（AGM不等式）以及牛顿二项式展开。

    3. 实数系的构造（附录 §1.4）：

      • 介绍了利用 Dedekind 分割从有理数构造实数的方法。
      • 通过分割定义了实数的序关系、加法和乘法运算，并证明了实数系的连续性。

    原文公式整理#

    按照原文公式后的序号整理如下：

    • (1.1) 等差数列求和： $1 + 2 + \dots + n = \frac{1}{2}n(n+1)$
    • (1.2) 求和号表示： $\sum_{k=1}^n k = \frac{1}{2}n(n+1)$
    • (1.3) 平方和公式： $\sum_{k=1}^n k^2 = \frac{1}{6}n(n+1)(2n+1)$
    • (1.4) 面积估算的误差控制： $|S - \frac{1}{3}| < \frac{1}{2n} + \frac{1}{6n^2} < \frac{1}{n}$
    • (1.6) 实数构造中的逼近估计（见附录）： $0 < 2 - x_{n+1}^2 < \frac{1}{4^n}, \quad n \ge 1$

    其他重要公式（源文中未加序号但核心）#

    • 几何级数求和：$\sum_{k=0}^{n-1} q^k = \frac{1-q^n}{1-q} \quad (q \neq 1)$
    • 最大值与最小值函数：
      • $\max{a, b} = \frac{1}{2}(a+b+|a-b|)$
      • $\min{a, b} = \frac{1}{2}(a+b-|a-b|)$
    • 三角不等式：$|a+b| \le |a| + |b|$
    • 均值不等式：$\sqrt{ab} \le \frac{a+b}{2} \quad (a, b \ge 0)$
    • 牛顿二项式公式：$(a+b)^n = \sum_{k=0}^n C_n^k a^k b^{n-k}$
